# Product

## Register

product

## Users
AI engineers and developers building, running, and evaluating DSPy-based agents and pipelines. Their context is local development, needing a secure environment to manage agent execution and human-in-the-loop interactions.

## Product Purpose
Vloop Harness serves as an AI-driven operating system sandbox. It provides a local-first workbench combining a native Rust kernel, a Python cognitive engine, and a React userland to securely execute and monitor AI agents.

## Brand Personality
Privacy, fluid, simple.

## Anti-references
Generic SaaS dashboards, cluttered layouts, or non-intuitive navigation structures.

## Design Principles
- **Privacy First**: Local-first execution should feel secure and self-contained.
- **Fluid Workflows**: Interactions between the user, the agent, and the sandbox must be seamless and responsive.
- **Simplicity over Complexity**: Hide the deep technical complexity of the three-tier architecture behind a clean, intuitive interface.

## Accessibility & Inclusion
Standard web accessibility baseline (keyboard navigation, sufficient contrast ratios).
